# graph-based-truth-set

This repository holds scripts for all analysis done for [this paper](), including truth set validation, model fitting, and figure generation. More information about can be found in Methods section of our paper. 

## File Structure
```markdown
├── alignmentBasedValidation/
    ├── [SNVs (mpileup)]
    └── [Indels and SVs (in-house cigarCaller)]
├── assemblyBasedValidation/
    ├── [Dipcall]
    └── [regionRescue code]
├── manuscriptFigures/
    └── [code to create all manuscript figures]
├── modelFitting/
    └── [code for model fitting]
├── simulations/
    ├── shortReadSimulation/
    └── longReadSimulation/
└── vafEvaluation/
```
